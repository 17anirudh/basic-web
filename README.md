
# Basic Website

Dynamic Project with Anchor Tags
This project is designed to showcase dynamic functionality using anchor tags. The primary features include URL redirection and the display of PDF files.


## Features

- Dark mode: By default this website uses dark mode.
- Redirection: Redirectes between different html files quickly.
- Minimal: Used Minimal in-line CSS.




## Run Locally

Clone the project

```bash
  git clone https://github.com/17anirudh/basic-web.git
```
Just preview it in the web 
```bash
https://17anirudh.github.io/basic-web/
```
