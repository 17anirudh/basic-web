Dynamic Project with Anchor Tags
This project is designed to showcase dynamic functionality using anchor tags. The primary features include URL redirection and the display of PDF files.

Features
1. URL Redirection
The project employs anchor tags (<a>) to demonstrate URL redirection. Users can navigate through the application by clicking on specified links, leading to different sections or pages.
2. Basic CSS attributes
This uses 2-3 in-line CSS attributes making it minimalized.
3. Displaying PDFs
Additionally, the project provides a mechanism to display PDF files directly through anchor tags. This is achieved by linking to the PDF file's location.

Getting Started:
To explore this project locally, follow these steps:
Clone the repository to your local machine:

git clone https://github.com/17anirudh/basic-web.git
Open the project in your preferred code editor.

Usage:
Simply open the index.html file in a web browser to experience the dynamic functionalities provided by the anchor tags.
open index.html
